# imail-web

The iMail landing page — a single static page served by GitHub Pages at
**https://lukasthedude.github.io/imail-web/**.

```
index.html   the page
styles.css   fluid layout (clamp/vw) — matches the 1920 design, scales to mobile
assets/      Inter font, app icon, Mac screenshot, favicon, link-preview image
.nojekyll    serve files as-is (no Jekyll processing)
```

The heavy "iMail" wordmark is Inter Black (900). "Join the Waitlist" opens the
visitor's mail app pre-addressed to lukas.lau@icloud.com.

No build step. Edit, commit, push — GitHub Pages redeploys automatically.
